Welcome to RIFT

We aren't looking for standard CRUD apps. RIFT 2026 is designed to push the boundaries of what's possible over a weekend. Tackle real-world problem statements provided by industry leaders and compete for massive equity-free grants.

Official Problem Statements

Choose your battlefield. You must select one of these four exact tracks to be eligible for judging.

HealthTech Track

PharmaGuard

Build an AI-powered web app that analyzes patient genetic data and drug names to predict personalized pharmacogenomic risks.

Key Requirements:

  • Parse authentic VCF v4.2 files.
  • Predict drug-specific risks across 6 critical genes (incl. CYP2D6 and TPMT).
  • Generate LLM-powered clinical explanations.
  • Strict output formatting in a defined JSON schema.
Graph Theory Track

Money Muling Detection

Build a web-based Financial Forensics Engine that processes transaction data to expose hidden money muling networks.

Key Requirements:

  • Accept CSV file uploads of transaction logs.
  • Provide interactive directed-graph visualizations.
  • Detect structural patterns: circular routing (cycles) & smurfing.
  • Flag layered shell networks and suspicious nodes.
AI / DevOps Track

Autonomous Healing Agent

Build an Autonomous DevOps Agent with a publicly deployed React Dashboard that detects, fixes, and verifies code pipeline issues.

Key Requirements:

  • Accept a target GitHub URL as input.
  • Automatically run tests and fix syntax/logic errors.
  • Commit fixes strictly to a new branch named TEAM_NAME_LEADER_NAME_AI_Fix.
  • Must use the specific [AI-AGENT] commit prefix.
Web3 Track

Build on Algorand

An open innovation challenge to build a real-world application (DeFi, NFTs, or DAO governance) meaningfully leveraging blockchain.

Key Requirements:

  • Must use AlgoKit as the primary development framework.
  • Deploy smart contracts strictly to the Algorand Testnet.
  • Submit a verifiable App ID with your project.
  • Ensure frontend integration with a compatible wallet.
